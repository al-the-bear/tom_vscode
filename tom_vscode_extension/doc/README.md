# DartScript VS Code Extension Documentation

This folder contains user-facing documentation for the `tom_vscode_extension` plugin.

## Core Documents

- `user_guide.md` — end-to-end feature usage guide.
- `quick_reference.md` — compact command and shortcut reference.
- `copilot_chat_tools.md` — Copilot and Ask-AI tooling reference (merged).

## YAML Graph Editor

- `yaml_graph.md` — current behavior and usage.
- `yaml_graph_architecture_design.md` — implementation architecture.
- `diagram_editing.md` — editing workflow and interaction model.

## Supporting References

- `file_and_prompt_placeholders.md` — supported placeholders and variable expansion.
- `reusable_component_analysis.md` — reusable webview/UI component inventory.

## Panel Naming (Current)

- Bottom panel `@CHAT` hosts the Unified Notepad webview (`dartscript.chatPanel`).
- Bottom panel `@WS` hosts the workspace webview (`dartscript.wsPanel`).
