# Tom Status Page

## Command

- `dartscript.showStatusPage`

## Purpose

Provide runtime visibility into extension state, configuration highlights, and operational checks.

## Expected content areas

- bridge/runtime status,
- AI subsystem readiness,
- configured paths/profile indicators,
- quick action links to key commands/settings.

## Maintenance notes

When adding a major subsystem:

1. expose summarized status in status page handler,
2. include actionable troubleshooting link or command,
3. keep output concise and human-readable.
